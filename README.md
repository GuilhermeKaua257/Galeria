# Galeria de Fotos

## Descrição do Projeto
Este projeto consiste em uma galeria de fotos estática, desenvolvida utilizando Vue.js, com foco na organização de layout e estruturação de componentes. A aplicação exibe imagens externas organizadas em formato de grid.

O projeto tem caráter educacional e foi desenvolvido para demonstrar conhecimentos iniciais em desenvolvimento front-end com Vue, HTML, CSS e JavaScript.

## Tecnologias Utilizadas
- JavaScript
- Vue.js
- HTML
- CSS
- Terminal / CLI

## Funcionalidades
- Exibição de galeria de imagens em grid
- Renderização de imagens externas via URL
- Exibição de títulos para cada imagem
- Cabeçalho com título da aplicação
- Campo de busca apenas visual (sem lógica implementada)
- Rodapé informativo

## Arquitetura
O projeto utiliza uma organização simples baseada em componente:

- App: componente principal responsável por toda a estrutura da aplicação
- Template: define o layout da galeria, cabeçalho e rodapé
- Estilos: responsáveis pela organização visual da galeria

A aplicação é totalmente front-end, sem backend.
